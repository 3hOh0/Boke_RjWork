# DjangoBlog功能扩展界面原型设计

## 1. 文章详情页（添加点赞/收藏功能）

### 1.1 页面布局

```
┌─────────────────────────────────────────┐
│           文章标题                      │
│           作者 | 发布时间 | 浏览量      │
├─────────────────────────────────────────┤
│                                         │
│           文章正文内容                  │
│                                         │
│                                         │
├─────────────────────────────────────────┤
│  [👍 点赞(123)]  [⭐ 收藏]  [💬 评论]   │
├─────────────────────────────────────────┤
│                                         │
│         相关文章推荐                    │
│  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │文章1 │  │文章2 │  │文章3 │          │
│  │摘要  │  │摘要  │  │摘要  │          │
│  └──────┘  └──────┘  └──────┘          │
└─────────────────────────────────────────┘
```

### 1.2 功能说明
- **点赞按钮**：点击后显示点赞数，已点赞状态高亮
- **收藏按钮**：点击后弹出收藏分类选择框
- **相关文章**：显示5篇推荐文章，卡片式布局

## 7. 界面设计说明

### 7.1 颜色方案
- **主色调**：蓝色（#007bff）
- **成功色**：绿色（#28a745）
- **警告色**：黄色（#ffc107）
- **错误色**：红色（#dc3545）

### 7.2 字体大小
- **标题**：18px-24px
- **正文**：14px-16px
- **辅助文字**：12px

### 7.3 交互效果
- 按钮悬停：颜色加深，添加阴影
- 卡片悬停：轻微上浮效果
- 加载状态：显示加载动画
- 操作反馈：成功/失败提示信息
